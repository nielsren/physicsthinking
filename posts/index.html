GithubPages教程

这是我用 Hugo 搭建的第三篇博客！

Hugo 网站更新与发布完整教程（PowerShell + 完整路径版）

第一步：打开 PowerShell

方法一（推荐）:

  1. 在 Hugo 项目文件夹任意位置按住 Shift + 右键
  2. 选择“在此处打开 PowerShell 窗口”

方法二:

  1. 按 Win + R
  2. 输入 powershell 回车

第二步：进入 Hugo 网站根目录

假设 Hugo 项目路径为： D:\HugoSites\myblog

PowerShell 输入:

cd D:\HugoSites\myblog

验证是否进入:

dir

应看到:

content
themes
layouts
static
config.toml
public

第三步：创建新的 Markdown 文章文件

Hugo 命令创建（推荐）:

hugo new post/first-post.md

文章路径:

D:\HugoSites\myblog\content\post\first-post.md

编辑文章内容（示例）:

---
title: "我的第一篇 Hugo 文章"
date: 2025-12-10
draft: false
---

这是我的第一篇博客文章。

注意: draft 必须设为 false 否则不会生成到 public。
